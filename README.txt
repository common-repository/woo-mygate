Accept payments by credit card, bank debit or banking ticket using the Mygate payment gateway.

== Description ==

This is a WooCommerce expension for the South African Mygate payment gateway. It supports payments from VISA, MasterCard, Diners Club and American Express credit card payments.

== Installation ==

1. Upload `woocommerce-mygate` to the `/wp-content/plugins/` directory (or search for Woocommerce Mygate directly from within the wordpress cms)
2. Activate the plugin through the 'Plugins' menu in WordPress
3. To start accepting payments via Mygate, add Woocommerce MyGate as one of the payment options by going to Woocommerce->Settings->Payments then Enable
Mygate.
4. Then click "Manage" and ensure the "Enable/Disable" checkbox is checked.
5. Specify your Mygate MerchantID and Application ID, both of which can be obtained from Mygate.
6. Uncheck "Enable Mygate Sandbox Mode" to start accepting payments. 


== Frequently Asked Questions ==

= Is the plugin for South Africa only? =

No. It works in any country where MyGate is available. This includes Botswana,
Ghana, Kenya, South Africa etc.

= How can I test it in Sandbox mode =

Ensure that the "Enable Mygate Sandbox Mode" checkbox is checked.
Use the Merchant ID: F5785ECF-1EAE-40A0-9D37-93E2E8A4BAB3
and Application ID: 3C5F80CB-F2DA-45D3-800D-A876E6258F17
Then use any of the test credit cards numbers available from the 
Mygate site: developers.mygateglobal.com/my_virtual.php
